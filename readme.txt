== Description ==

DISCLAIMER: Fat Panda, LLC is not affiliated with Singly, Inc.

This plugin helps you use [AppFabric by Singly](http://singly.com) to allow your users to log into
your WordPress site using their accounts with Facebook, Twitter, Google, and 29 other social-ready services.

== Installation ==

1. Get the plugin. Activate the plugin.

2. [Create a Singly account](https://singly.com/signup). Their free plan supports up to 100 user accounts.

3. Create a Singly App. 

4. Go to **Settings / General**, fill in your **Client ID** and **Client Secret** keys, and select the services you're supporting. 

5. Test by logging out of WordPress, and then visitng the login screen. You should see login options for each service.

6. Once satisified it's working, [setup keys on singly.com](https://singly.com/apps) for each of the services you want to support.

Need support with Singly? [Go here](https://singly.com/docs/contact).
